Acceleration2D Class

Terminology

None.

ConOps

Represent the abstract concept of a 2-dimensional acceleration in cartesian space. The internal representation is made up of 2 Acceleration class variables, X & Y. Functions include comparison and arithmetic operators. A function is provided to determine the magnitude of a object of Acceleration2D. Functions are provided to determine between two Acceleration2D objects a midpoint, interpolate, distance, and angle. Functions provided to read and write an acceleration object in binary format. A function providing a self-test is provided.

Interfaces with this class are shown at the Unit Library Service level.

Implementation Details

Acceleration2D is one class in a set of classes providing basic unit objects needed for a variety of applications.

Object Scenarios / Test Procedures

An object of Acceleration2D can be declared without a value. When a value is assigned to that object, it must be defined in terms of either a set of 2 Acceleration values or another Acceleration2D object.

To test this class, simply run the unitsTest.exe program by typing "unitsTest Acceleration2D". The test routine will indicate if the class passed it's self-test.

Example Code Snippet

      Acceleration2D accelerationVector = (MetersSecondAcceleration(1),
                                           MetersSecondAcceleration(1));

      Acceleration   acceleration = accelerationVector.magnitude();

      cout << "acceleration = " << acceleration << endl;
