SignedAngle Class

Terminology

None.

ConOps

Represent the abstract concept of a signed circular angle (values wrap around). The internal representation of the units is inherited from the Angle class. Most functions are inherited from the Angle class. Functions to construct, convert to a double, and the stream operator are overwritten so that the angle can be normalized if needed. A function providing a self-test is provided. A set of common subclasses are provided.

Interfaces with this class are shown at the Unit Library Service level.

Implementation Details

SignedAngle is one class in a set of classes providing basic unit objects needed for a variety of applications.

When a value is assigned to a SignedAngle object, the value is converted into the internal representation units as defined by the Angle class. When a value is requested from an object, it is converted into the requested units.

When a SignedAngle object is converted to a double, the value is normalized to be between 0 and 360 degrees or between 0 and 2pi radians, depending on the subclass specified.

A circular angle wraps around; that means DegreesAngle(370) == DegreesAngle(10).

Subclasses defined are SignedDegreesAngle and SignedRadiansAngle.

Object Scenarios / Test Procedures

An object of SignedAngle can be declared without a value. When a value is assigned to that object, it must be defined in terms of a subclass to SignedAngle. When an object of a SignedAngle class is declared and is given a value, that value must be specified in terms of some pre-defined subclass. This is the only way that the SignedAngle class can know how the number provided should be interpreted.

A function is provided to convert a SignedAngle object to a double. This also requires a subclass specification so the caller can be assured of getting the result in an expected way that is not dependent on the internal representation.

To test this class, simply run the unitsTest.exe program by typing "unitsTest SignedAngle". The test routine will indicate if the class passed it's self-test.
