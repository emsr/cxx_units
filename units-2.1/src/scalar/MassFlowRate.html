MassFlowRate Class

Terminology

None.

ConOps

Represent the abstract concept of a speed. Functions include comparison, arithmetic, and interaction operators. Methods provided when necessary to interact with related classes. Functions provided to read and write an MassFlowRate object in binary format. A function providing a self-test is provided. A set of common subclasses are provided.

Interfaces with this class are shown at the Unit Library Service level.

Implementation Details

MassFlowRate is one class in a set of classes providing basic unit objects needed for a variety of applications.

Subclasses defined for convenience are PoundsPerHourMassFlowRate and KilogramsPerHourMassFlowRate.

Object Scenarios / Test Procedures

An object of MassFlowRate can be declared without a value. When a value is assigned to that object, it must be defined in terms of a Length subclass and a Time subclass. When an object of a MassFlowRate class is declared and is given a value, that value must also be specified in terms of a Length subclass and a Time subclass. This is the only way that the MassFlowRate class can know how the value provided should be interpreted.

A function is provided to convert a MassFlowRate object to a double. This requires a Length subclass and a Time subclass specification so the caller can be assured of getting the result in an expected way that is not dependent on the internal representation.

To test this class, simply run the unitsTest.exe program by typing "unitsTest MassFlowRate". The test routine will indicate if the class passed it's self-test.
