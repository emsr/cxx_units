AngularSpeed Class

Terminology

None.

ConOps

Represent the abstract concept of angular speed, which is the rate of change of an angle. The internal representation of the units is hidden from the user. Functions include comparison, arithmetic, interaction, and trigonometric operators. Methods provided when necessary to interact with related classes. Functions provided to read and write an angularspeed object in binary format. A function providing a self-test is provided.

Interfaces with this class are shown at the Unit Library Service level.

Implementation Details

AngularSpeed is one class in a set of classes providing basic unit objects needed for a variety of applications.

Object Scenarios / Test Procedures

An object of AngularSpeed can be declared without a value. When a value is assigned to that object, it must be defined in terms of a Angle subclass and a Time subclass. When an object of a AngularSpeed class is declared and is given a value, that value must also be specified in terms of a Angle subclass and a Time subclass. This is the only way that the AngularSpeed class can know how the value provided should be interpreted.

A function is provided to convert a AngularSpeed object to a double. This also requires a subclass specification so the caller can be assured of getting the result in an expected way that is not dependent on the internal representation.

To test this class, simply run the unitsTest.exe program by typing "unitsTest AngularSpeed". The test routine will indicate if the class passed it's self-test.

Example Code Snippet

      Angle changeHeading (const Angle & currentHeading,
                           const Time  & duration)
      {
        // for example purposes, declare a constant rate of heading change
        AngularSpeed headingChangeRate = DegreesAngle(2.3)/SecondsTime(1.0);

        Angle newHeading = currentHeading + headingChangeRate*duration;

        return newHeading;
      }

Requirements

Provide comparison operators, as appropriate, to compare to objects of this class.

Provide arithmetic operators, as appropriate, to add, subtract, multiply, and divide objects of this class.

Provide operators to interact appropriately with the Time class, the Angle class, and doubles.

Provide a way to print the value of an object.

Provide binary read and write functions.

System Dependencies
