Pressure Class

Terminology

None.

ConOps

Represent the abstract concept of pressure. The internal representation of the units is hidden from the user. Functions include comparison and arithmetic operators. Functions provided to read and write an Pressure object in binary format. A function providing a self-test is provided. A set of common subclasses are provided.

Implementation Details

Pressure is one class in a set of classes providing basic unit objects needed for a variety of applications.

When a value is assigned to a Pressure object, the value is converted into the internal representation units. When a value is requested from an object, it is converted into the requested units.

Subclasses defined for convenience are AtmospheresPressure, InchesHgPressure, and PsiPressure.

Object Scenarios / Test Procedures

An object of Pressure can be declared without a value. When a value is assigned to that object, it must be defined in terms of a subclass to Pressure or as a Force devided by an Area. When an object of a Pressure class is declared and is given a value, that value must be specified in terms of some pre-defined subclass. This is the only way that the Pressure class can know how the number provided should be interpreted.

A function is provided to convert a Pressure object to a double. This also requires a subclass specification so the caller can be assured of getting the result in an expected way that is not dependent on the internal representation.

To test this class, simply run the unitsTest.exe program by typing "unitsTest Pressure". The test routine will indicate if the class passed it's self-test.

Requirements

Provide comparison operators, as appropriate, to compare to objects of this class.

Provide arithmetic operators, as appropriate, to add, subtract, multiply, and divide objects of this class.

Provide operators to interact appropriately with doubles.

Provide a way to print the value of an object.

Provide binary read and write functions.

System Dependencies

Required classes include:

  • Unit
    • This class inherits from the Unit class.
  • Force
    • Force is used in Pressure operators.
  • Area
    • Area is used in Pressure operators.
