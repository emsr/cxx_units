Speed3D Class

Terminology

None.

ConOps

Represent the abstract concept of a 3-dimensional speed in cartesian space. The internal representation is made up of 3 Speed class variables, X, Y, & Z. Functions include comparison and arithmetic operators. A function is provided to determine the magnitude of a object of Speed3D. Functions are provided to determine between two Speed3D objects a midpoint, interpolate, distance, and angle. Functions provided to read and write an speed object in binary format. A function providing a self-test is provided.

Interfaces with this class are shown at the Unit Library Service level.

Implementation Details

Speed3D is one class in a set of classes providing basic unit objects needed for a variety of applications.

Object Scenarios / Test Procedures

An object of Speed3D can be declared without a value. When a value is assigned to that object, it must be defined in terms of either a set of 3 Speed values or another Speed3D object.

To test this class, simply run the unitsTest.exe program by typing "unitsTest Speed3D". The test routine will indicate if the class passed it's self-test.

Example Code Snippet

      Speed3D speedVector = 
                 (MetersPerSecondSpeed(1),
                  MetersPerSecondSpeed(1),
                  MetersPerSecondSpeed(1));

      Speed   speed = speedVector.magnitude();

      cout << "speed = " << speed << endl;

Provide a way to print the value of an object.

Provide binary read and write functions.
