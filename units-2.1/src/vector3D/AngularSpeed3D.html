AngularSpeed3D Class

Terminology

None.

ConOps

Represent the abstract concept of a 3-dimensional angular speed in cartesian space, which is the rate of change of an angle. The internal representation is made up of 3 AngularSpeed class variables, X, Y, & Z. Functions include comparison and arithmetic operators. A function is provided to determine the magnitude of a object of AngularSpeed3D. Functions are provided to determine between two AngularSpeed3D objects a midpoint, interpolate, distance, and angle. Functions provided to read and write an angular speed object in binary format. A function providing a self-test is provided.

Interfaces with this class are shown at the Unit Library Service level.

Implementation Details

AngularSpeed3D is one class in a set of classes providing basic unit objects needed for a variety of applications.

Object Scenarios / Test Procedures

An object of AngularSpeed3D can be declared without a value. When a value is assigned to that object, it must be defined in terms of either a set of 3 AngularSpeed values or another AngularSpeed3D object.

To test this class, simply run the unitsTest.exe program by typing "unitsTest AngularSpeed3D". The test routine will indicate if the class passed it's self-test.

Example Code Snippet

      AngularSpeed3D angularSpeedVector = 
                 (DegreesAngle(1.0)/SecondsTime(1.0),
                  DegreesAngle(1.0)/SecondsTime(1.0),
                  DegreesAngle(1.0)/SecondsTime(1.0));

      AngularSpeed   angularSpeed = angularSpeedVector.magnitude();

      cout << "angularSpeed = " << angularSpeed << endl;
